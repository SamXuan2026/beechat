# BeeChat 后续迭代计划

## 第三期：工程化与部署

- 前端迁移到 Vite + React 或 Vue。
- 引入组件化状态管理。
- 补充数据库迁移执行器。
- 增加 Docker 镜像版本号。
- 增加接口自动化测试。
- 增加消息分页。

## 第四期：企业协同增强

- 频道公告。
- 置顶消息。
- @ 人提醒。
- 消息收藏。
- 图片预览。
- 文件列表。
- 全局搜索结果定位。

## 第五期：安全与管理

- 管理后台。
- 审计导出。
- IP 白名单。
- 密码策略。
- 操作水印。
- 数据备份与恢复。

## 第六期：生产级架构

- PostgreSQL 适配。
- Redis 会话与 WebSocket 分布式广播。
- 对象存储适配。
- 单点登录。
- 组织架构同步。
