# 设计文件归档

本目录统一存放 BeeChat / BeePilot 相关设计产物。

## 目录规则

```text
design/beechat/   BeeChat 聊天工作台、频道、线程、审计、成员等界面设计
design/beepilot/  BeePilot Admin、Widget、客户会话、线索中心等界面设计
```

## 文件类型

建议归档：

- `.pen` Pencil 设计源文件
- `.fig` Figma 源文件或导出说明
- `.png` / `.jpg` 截图
- `.pdf` 设计评审稿
- `.md` 设计说明
