# BeeChat 设计稿

本目录用于存放 BeeChat 相关设计文件。

建议命名：

```text
beechat-main-chat.pen
beechat-thread-view.pen
beechat-admin-users.pen
beechat-design-review-YYYYMMDD.png
```

当前实现参考的核心界面：

- 主聊天页
- 线程面板
- 审计面板
- 成员面板
- 登录页

## 已归档文件

```text
DESIGN_SPEC.md  BeeChat 多画板重建设计规范
beechat-main-chat.pen  BeeChat 多画板 Pencil 正式源文件
```

## 历史占位文件

```text
beechat-main-chat.placeholder-old.pen  历史 pencil-new.pen 旧占位文件
```

注意：`beechat-main-chat.placeholder-old.pen` 是从历史 `pencil-new.pen` 复制的旧占位文件，内容仍是“多模型智能聊天工作台”，不可作为 BeeChat 设计源使用。正式源文件重建完成后应命名为 `beechat-main-chat.pen`。
