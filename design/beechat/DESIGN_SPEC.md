# BeeChat 设计规范

## 设计状态

BeeChat 原 Pencil 多画板源文件已无法在本机可读目录中定位。当前设计以历史截图和 BeeChat MVP 已实现功能为基准重建，后续所有 Pencil / Figma / 前端实现应以本文档为准。

## 设计目标

- 面向企业内网私有化协同聊天场景。
- 保持轻量、清晰、低学习成本的工作台风格。
- 优先支撑频道协作、私聊、线程、审计、搜索、文件、成员管理。
- 视觉上延续截图中的深紫侧边栏、浅色内容区、蓝色主操作按钮。

## 核心画板

### BeeChat - Main

主聊天工作台，尺寸建议 `1280 x 720`。

布局：

- 左侧导航栏，宽度 `232`，背景 `#281850`。
- 顶部品牌区：BeeChat 标识、当前工作区下拉。
- 搜索框：深紫输入框，支持搜索频道、成员、消息。
- 频道列表：`# general`、`# random`、`# engineering`、`announcements`。
- 私信列表：张三、李四、王五，在线状态用绿色圆点。
- 底部当前用户卡片：头像、昵称、更多操作。
- 中央消息区：频道标题、成员数、工具图标、内网审计状态。
- 消息列表：日期分割线、系统安全提示、成员消息、本人消息、审计提示。
- 底部输入区：富文本按钮、附件、输入框、表情、审计标识、发送按钮。

关键状态：

- 当前频道高亮：`#5A3A9E`。
- 未读数：红色圆形徽标 `#E05252`。
- 本人消息气泡：蓝色 `#4A9FD8`，右侧对齐。
- 他人消息：无卡片或浅色文本块，左侧头像。
- 审计提示条：蓝色信息条与橙色敏感词提示条。

### BeeChat - Thread View

线程页，尺寸建议 `1280 x 720`。

布局：

- 继承主聊天左侧导航和中央消息区。
- 右侧线程面板，宽度 `320`，背景 `#FFFFFF`，左边框 `#E6EAF2`。
- 线程标题：线程图标、标题、关闭按钮。
- 根消息卡片：展示原消息、作者、时间、线程回复数。
- 回复列表：按时间排列，支持审计状态和敏感提示。
- 底部线程回复输入框。

关键状态：

- 当前线程消息在中央列表中保持可识别。
- 线程面板内敏感提示使用浅橙背景 `#FFF5DF`。
- 回复数量使用小号蓝色文本。

### BeeChat - Login

登录页，尺寸建议 `720 x 540`。

布局：

- 浅灰背景 `#F5F7FB`。
- 居中登录卡片，宽度 `360`，圆角 `10`。
- 顶部品牌图标：蓝色方块 `B`。
- 标题：登录 BeeChat。
- 字段：账号、密码。
- 主按钮：登录。
- 辅助入口：语言、消息、忘记密码、立即注册。

关键状态：

- 输入框背景 `#F6F7FA`。
- 登录按钮背景 `#4A9FD8`。
- 错误态使用 `#E05252`，需要保留字段级错误提示位置。

### BeeChat - Discover Channels

发现频道页，尺寸建议 `720 x 540`。

布局：

- 顶部栏：返回按钮、标题“发现频道”、创建频道按钮。
- 搜索框：搜索频道。
- 频道卡片列表：图标、频道名、描述、加入/已加入按钮。
- 频道：general、engineering、design、random。

关键状态：

- 已加入按钮使用绿色 `#41C585`。
- 未加入按钮使用蓝色 `#4A9FD8`。
- 频道图标按类型使用蓝、橙、绿区分。

### BeeChat - Settings

设置页，尺寸建议 `840 x 620`。

布局：

- 左侧设置导航，宽度 `208`。
- 导航项：个人资料、通知、外观、快捷键。
- 右侧内容区：个人资料表单。
- 头像、显示名称、邮箱、个人简介、保存按钮。

关键状态：

- 当前导航高亮背景 `#EEF1F6`。
- 保存按钮使用蓝色主按钮。
- 表单输入框统一浅灰背景，字号 `13`。

| 设计区域 | 对应 MVP 能力 |
| --- | --- |
| 登录页 | 登录态、本地账号 |

| 消息列表 | 消息分页、发送、编辑、撤回 |
| 本人消息 | 已读状态、发送状态 |
| 线程面板 | 线程回复 |
| 搜索框 | 消息搜索 |
| 附件入口 | 文件上传、下载 |
| 审计提示 | 敏感词、操作审计 |
| 设置页 | 用户资料、后续偏好设置 |
